# uniquid-cli [![Build Status][travis-image]][travis-url]

> Unique ID generator

[uniquid][uniquid] - API for this module

## Install

```sh
npm install --global uniquid-cli
```

## Usage

```sh
uniquid --help

  Usage
    uniquid [prefix]

  Examples
    uniquid
    uniquid hello
```
