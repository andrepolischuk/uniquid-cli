# uniquid-cli

  > Unique ID generator

  [uniquid][uniquid] - API for this module

## Install

```sh
npm install --global uniquid-cli
```

## Usage

```sh
uniquid --help

  Usage: uniquid [prefix]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number

  Examples:

    # Unique ID
    uniquid

    # Prefixed unique ID
    uniquid hello
```
